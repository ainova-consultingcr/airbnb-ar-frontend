# Company Ops - AVI / Consultoria IA

Este directorio es el sistema operativo liviano para manejar la empresa mientras trabajas solo.

La idea no es crear burocracia. La idea es que cada archivo reduzca carga mental y te ayude a mover tres frentes:

- Impulsar AVI como producto principal.
- Dar a conocer la empresa de consultoria IA.
- Convertir conversaciones con prospectos en pilotos, propuestas o proyectos a medida.

## Ritmo semanal sugerido

### Lunes - Direccion
- Revisar `backlog_avi.md`.
- Elegir 3 prioridades maximas de la semana.
- Revisar `pipeline_clientes.md`.
- Definir 5 a 10 prospectos para contactar.

### Martes a jueves - Ejecucion
- Desarrollo de AVI.
- Contacto comercial.
- Publicaciones de contenido.
- Demos o conversaciones con prospectos.

### Viernes - Cierre
- Actualizar `feedback_piloto.md`.
- Registrar aprendizajes.
- Crear resumen semanal.
- Preparar contenido de la proxima semana.

## Agentes internos

Los prompts de agentes estan en `agents/`.

Uso recomendado:

1. Copia el prompt del agente.
2. Pegalo en una conversacion nueva o usalo conmigo.
3. Dale contexto real: avances, bugs, cliente, idea o feedback.
4. Pidele salida concreta: tareas, propuesta, post, checklist o plan.

## Regla de foco

Durante la primera etapa, AVI es el producto bandera.

La empresa puede vender proyectos a medida, pero todo debe reforzar una posicion:

> Creamos asistentes inteligentes y automatizaciones para negocios que quieren atender mejor, vender mas y operar con menos friccion.
